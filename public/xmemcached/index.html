High performance, easy to use multithreaded memcached client in java

Features:

  • Full feature of text/bianry protcol for memcached
  • Synchronous API
  • Multithreaded/thread-safe
  • Multiple server nodes with consistent hash or standard hash
  • Use JMX to montior and control
  • Weighted server supports
  • Add/Remove server or change server's weight dynamically
  • Client statistics
  • Auto reconnection and heartbeat to keepalive
  • Built-in spring and hibernate-memcached integration
  • Nio connection pool
  • Failure mode and standby nodes
  • Support kestrel
